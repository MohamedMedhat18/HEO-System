## Fonts Used in the Application

### Arabic Font
- **Tajawal**: A modern Arabic typeface that is clean and easy to read. It is used throughout the application for all Arabic text to ensure clarity and professionalism.

### English Fonts
- **Lato**: A sans-serif typeface that is known for its modern and friendly appearance. It is used for English text in the application.
- **Roboto**: Another popular sans-serif typeface that offers a versatile and professional look. It can be used interchangeably with Lato for English text.

### Usage
To use these fonts in the application, ensure that they are properly imported in the CSS files. The fonts can be linked from Google Fonts or included in the project assets.
